# 🎬 影片與字幕自動合併工具 (自適應解析度版)

這是一個基於 **Gradio 6.x** 與 **FFmpeg** 核心開發的線上影片硬壓字幕工具，專門解決 Linux 伺服器環境下常見的「中文字型缺失（口口口）」與「低解析度影片字體肥大」等痛點。

## ✨ 核心特色

1. **⏱️ 兩分鐘快速試看機制 (新增)**
   - 提供「測試合併」按鈕，可在幾秒鐘內火速產出影片前 2 分鐘的渲染效果。
   - 調整字體大小時無需再等待整部影片漫長轉碼，滿意後再一鍵進行正式完整合併。
2. **🧠 ffprobe 解析度智能感應**
   - 後端會自動感應上傳影片的實際垂直像素（高度）。
   - 以標準 1080p 為基準，自動等比例縮放字幕像素大小（FontSize），不論影片大或小，都能完美維持你設定的視覺比例，絕不肥大。
3. **🚫 徹底根除「口口口」方塊字**
   - 程式啟動時會在系統底層無痛補強 Linux 系統級中文字型包（Google 思源黑體 `fonts-noto-cjk`），從根源解決中文字型缺失問題。
4. **🧹 字幕樣式強制純淨化覆寫**
   - 自動剝離原始字幕（SRT/ASS）檔案中夾帶的頑固內置樣式或 HTML 標籤（如 `<font>`, `{\pos...}` 等），100% 確保前端控制台設定的字體尺寸精準生效。
5. **🔒 版面鎖定不變形**
   - 嚴格鎖定 Gradio 影片組件顯示範圍（高度 360px）。影片導入或產生後，皆會乖乖待在畫出來的格子空間內，絕不放大撐開或破壞網頁版面。

## 🛠️ 參數內定說明

- **中文/雙語字幕基準大小**：內定 `20`（以 1080p 影片為基準，小影片會自動等比縮小）。
- **純外文字幕基準大小**：內定 `12`（以 1080p 影片為基準，適合精巧的英/日/韓文字幕）。
- **字幕外觀**：精準配置 1.0 像素細緻黑邊（Outline），不帶突兀陰影，並隨影片解析度自動縮放垂直留白（MarginV）

📝 使用指南
上傳影片與字幕：將你的原始影片（MP4 / MKV）與字幕檔（SRT / ASS）分別拖入對應的格子。

微調尺寸：根據需求保持預設值（20 / 12）或手動調整滑桿。

快速測試：點擊 ⏱️ 測試合併 (僅前2分鐘)，等待數秒後在右側預覽字體大小與畫面比例是否滿意。

正式合併：確認滿意後，點擊 🚀 開始正式完整合併 壓製全片，完成後即可直接線上播放或點擊右上角下載成果。
